# АНО «МЦМС» — Сайт

Одностраничный сайт [Московского центра международного сотрудничества](https://ano-mcms.ru).

## Структура

- `index.html` — полный сайт (HTML + CSS + JS в одном файле)

## Технологии

- Семантический HTML5, ARIA-разметка
- CSS Custom Properties, CSS Grid, Flexbox
- Vanilla JS (ES5) — без зависимостей
- Адаптивность: 1280 / 1024 / 768 / 480 px (desktop-first)
- Шрифты: PT Serif + PT Sans (Google Fonts)

## Публикация через GitHub Pages

1. Перейдите в **Settings → Pages** репозитория
2. Source: `Deploy from a branch`
3. Branch: `main`, папка `/ (root)`
4. Сохраните — сайт будет доступен по адресу `https://<user>.github.io/<repo>/`
